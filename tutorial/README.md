# AI × BDD/DDD 入門チュートリアルシリーズ

## このチュートリアルについて

AIを使った開発プロセスを、3つのサンプルプロジェクトを通じて段階的に学ぶシリーズ。
各チュートリアルは前のチュートリアルの上に積み重なる設計になっており、
最終的に「振る舞い定義を起点にしたAIとの協調開発」を実践できるようになることを目指す。

リファレンスとして `guide/` のドキュメント群を参照しながら進めること。

---

## 対象読者

- AIを使い始めたばかりの開発者
- BDD・DDDに興味はあるが実践経験が少ない開発者
- AIと協調した開発プロセスを体系的に学びたい開発者

---

## 全体構成

| | チュートリアル1 | チュートリアル2 | チュートリアル3 |
|--|--------------|--------------|--------------|
| **テーマ** | Kanbanボード | レストラン注文システム | 予約システム |
| **主な学習内容** | AI開発フローの全体体験 | BDD・DDDの基礎 | ドメインルール・制約・コンテキスト分割 |
| **新しく導入するプラクティス** | 実例マッピング・要求定義・詳細設計・実装の流れ | イベントストーミング・ポリシー・発見手法のサイクル | 集約・競合制御・境界付けられたコンテキスト |
| **想定規模** | 1人・単機能 | 小チーム想定 | 中規模想定 |
| **難易度** | ★☆☆ | ★★☆ | ★★★ |

---

## 前提知識

- プログラミングの基礎知識
- ターミナル操作の基礎
- コーディングAIのインストール済み

BDD・DDD・イベントストーミングの事前知識は不要。チュートリアルの中で説明する。

## 推奨する進め方

1. `guide/01_philosophy.md` を読んで全体思想を理解する
2. チュートリアル1（Kanban）を最初から最後まで通しでやる
3. チュートリアル2（レストラン）へ進む
4. チュートリアル3（予約）で全プラクティスを統合する
